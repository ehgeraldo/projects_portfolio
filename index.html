Projetos em Ciência de Dados

Previsão de vendas para rede de farmácias

'Rossmann' administra mais de 3.000 drogarias em 7 países europeus. Como critério de liberação de recursos para cada unidade houve processo de escolha para disponibilizar tais recursos para cada unidade específica:

Os gerentes de loja da Rossmann têm a tarefa de prever suas vendas diárias com até seis semanas de antecedência.


As vendas são influenciadas por inúmeros fatores, incluindo promoções, competição entre lojas próximas e afastadas, feriados estaduais, sazonalidade e localidade.


Com milhares de gerentes individuais prevendo vendas com base em suas circunstâncias únicas, a precisão dos resultados pode ser bastante variada.


Ferramentas utilizadas

  • Python, Pandas, Numpy, Seaborn, Sklearn, Boruta.
  • Git, Github, Virtual Environment, Linux.
  • Previsão com Regressão através de machine Learning.
  • Cloud Heroku.

Identificação de imóveis para compra e revenda

A 'House Rocket' é uma plataforma digital com modelo de negócio atribuído à compra e a venda de imóveis utilizando dados como critério de aquisição.

O objetivo é encontrar as melhores oportunidades de negócio no mercado de imóveis.


A principal estratégia é indicar os imóveis com ótimas localizações e preços baixos e atrativos para maximizar a revenda e a receita.

Identificação de imóveis abaixo do preço médio de venda e definição do preço ideal de revenda, a partir de análise explorátoria de dados em Python.

Ferramentas utilizadas

  • Python, Pandas, Numpy, Seaborn, Plotly,Sklearn
  • Git, Github, Virtual Environment, Linux.

Representação de dados de poços de petróleo

A visualização de dados de poços de petróleo auxilia a tomada de decisão em vários contextos (geologia, petrofísica, direcionamento de poços).

O objetivo é aplicar diferentes bibliotecas nos dados abertos de diferentes poços.


A principal estratégia é demonstrar de maneira simples como segue a sequência de visualização de dados no segmento utilizando python.

Identificação de regiões promissoras com análise visual são processos impressindíveis no segmento, a partir de análise explorátoria de dados com Python.
